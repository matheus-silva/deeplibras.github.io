    Brazilian Sign Language Interpretation using Machine Learning Techniques

    The DeepLIBRAS is a research project that aim at developing an Artificial Intelligence (IA) able to interpret the Brazilian Sign Language (LIBRAS). We hope this project become the deaf and hard of hearing community and communities that do not speak LIBRAS closer.

    To this aim we apply a set of statistic and machine learning techniques like: i) Maximal Information Coefficient; ii) Multilayer Perceptron Neural Network; iii) Recurrent Neural Network; iv) Deep Learning; v) Hidden Markov Models, dentre outras.
