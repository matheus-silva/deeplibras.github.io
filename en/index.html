    Brazilian Sign Language Interpretation using Machine Learning Techniques

    DeepLIBRAS is a research project that aims at developing an Intelligent Agent (IA) able to interpret the Brazilian Sign Language (LIBRAS). We hope this project will reduce the distance between the community of deaf and hard of hearing and communities that do not speak LIBRAS.

    In order to achieve this goal, we apply a set of statistic and machine learning techniques, such as: i) Maximal Information Coefficient; ii) Multilayer Perceptron Neural Network; iii) Recurrent Neural Network; iv) Deep Learning; v) Hidden Markov Models, among others.
